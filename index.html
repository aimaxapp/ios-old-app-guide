⚠️ 千萬不要升級 iOS！一旦升級到 17.0.1 以上，TrollStore 將永遠無法安裝。

1 安裝 TrollStore 巨魔商店

  1. 點擊下方按鈕，Safari 會提示安裝描述檔，點「允許」
  2. 前往「設定 → 一般 → VPN 與裝置管理」安裝描述檔
  3. 回到主畫面，打開新出現的「GTA Car Tracker」App
  4. 點擊「Install TrollStore」
  5. 安裝完成後，打開 TrollStore → Settings → Install Persistence Helper → 選「提示 (Tips)」
安裝 TrollHelperOTA 點擊後 Safari 會提示安裝描述檔

2 安裝 App 降級工具

下載後選擇「用 TrollStore 打開」即可安裝。三選一即可：

下載 MuffinStore v1.1 最推薦 28.6 KB · 開源免費 下載 DowngradeApp v1.1-8 備選 免 FaceID/TouchID 驗證 下載 AppStore++ v1.0.3-2 16.8 MB · 經典老牌工具

3 安裝 TrollFools（鎖版本）

安裝後可鎖定 App 版本，阻止強制更新彈窗。

下載 TrollFools v4.3 最新版 · 2026-04-16 更新

4 用 MuffinStore 下載舊版 Uber 重要

💡 MuffinStore 沒有搜尋功能！你需要「貼上 App Store 連結」才能下載。請按照以下步驟操作：
  1. 先複製下方的 Uber App Store 連結（點「複製」按鈕）
  2. 打開 MuffinStore App
  3. 點擊「Download」
  4. 在彈出的「App Link」對話框中，長按輸入框 → 貼上剛才複製的連結
  5. 點擊「Download」
  6. 選擇「Get version list from server」（從伺服器取得版本列表）
  7. 從列表中選擇一個舊版本安裝
  8. 系統會要求輸入 Apple ID 驗證，登入後即可下載

⚠️ 注意：下載過程沒有進度條，請耐心等待！安裝提示出現後點「安裝」，等 5 秒再回到主畫面。

如果想保留 App 資料，請先到「設定 → 一般 → iPhone 儲存空間」找到該 App，選「卸載 App」（不是刪除），再用 MuffinStore 降級安裝。

5 鎖定 App 版本（防強制更新）

  1. 打開 TrollFools
  2. 在列表中找到 Uber（或你降級的 App）
  3. 長按該 App → 點「鎖定版本」
  4. 回到 TrollStore → 點「重建圖標緩存」
  5. 前往「設定 → App Store」→ 關閉「App 更新」

6 iOS 版本偽裝（繞過系統檢查）進階

如果 Uber Eats 顯示「我們已不再支援您的作業系統」，表示 App 檢測到你的 iOS 版本太舊。用以下插件可以偽裝 iOS 版本號，讓 App 以為你是 iOS 18/19。

💡 原理：v3 全面版覆蓋所有已知的 iOS 版本檢測方式，包括底層 C 函數（sysctl、uname）和網路請求 User-Agent。純 C + ObjC Runtime，不依賴 CydiaSubstrate。
⚠️ 重要：請先移除之前注入的所有舊版插件！
打開 TrollFools → 找到 Uber Eats → 點「Plug-Ins」→ 移除所有舊插件 → 再注入 v3。
  1. 點擊下方按鈕下載 VersionSpoofer_v3.dylib
  2. Safari 會提示無法預覽，點「更多...」→「儲存到檔案」存到 iPhone
  3. 打開 TrollFools
  4. 在列表中找到 Uber Eats
  5. 先點「Plug-Ins」移除所有舊插件
  6. 點綠色的「Inject」按鈕
  7. 在檔案選擇器中找到剛下載的 VersionSpoofer_v3.dylib
  8. 選擇檔案後點「Open」
  9. 等待注入完成，完全關閉 Uber Eats（從多工列表上滑移除）
  10. 重新打開 Uber Eats 測試
下載 VersionSpoofer v3（全面版）⭐ 推薦 67 KB · arm64 · 偽裝為 iOS 18.4.1 · 覆蓋所有檢測方式 舊版 v2（備用） 65 KB · 僅覆蓋 ObjC API
⚠️ 注意事項：
・如果注入後 App 閃退，回到 TrollFools → 找到 Uber Eats → 點「Plug-Ins」→ 移除該插件
・也可以注入到其他 App（如 Uber 叫車、LINE 等）來繞過版本限制
・如果 v3 仍無法繞過，表示 Uber 的檢查完全在伺服器端進行，建議改用網頁版
插件名稱VersionSpoofer v3（全面版）
來源自製 · 純 C + ObjC Runtime + DYLD Interpose
