iOS 舊版 App 安裝指南

適用 iPhone XR · iOS 15.3.1 · 免越獄

📋 你的裝置資訊

機型iPhone XR
晶片A12 Bionic (arm64e)
iOS 版本15.3.1
安裝工具TrollHelperOTA
需要電腦不需要 ✅
⚠️ 千萬不要升級 iOS！一旦升級到 17.0.1 以上，TrollStore 將永遠無法安裝。

1 安裝 TrollStore 巨魔商店

  1. 點擊下方按鈕，Safari 會提示安裝描述檔，點「允許」
  2. 前往「設定 → 一般 → VPN 與裝置管理」安裝描述檔
  3. 回到主畫面，打開新出現的「GTA Car Tracker」App
  4. 點擊「Install TrollStore」
  5. 安裝完成後，打開 TrollStore → Settings → Install Persistence Helper → 選「提示 (Tips)」
安裝 TrollHelperOTA 點擊後 Safari 會提示安裝描述檔

2 安裝 App 降級工具

下載後選擇「用 TrollStore 打開」即可安裝。三選一即可：

下載 MuffinStore v1.1 最推薦 28.6 KB · 開源免費 · 介面簡潔 下載 DowngradeApp v1.1-8 備選 免 FaceID/TouchID 驗證 下載 AppStore++ v1.0.3-2 16.8 MB · 經典老牌工具

3 安裝 TrollFools（鎖版本）

安裝後可鎖定 App 版本，阻止強制更新彈窗。

下載 TrollFools v4.3 最新版 · 2026-04-16 更新

4 降級安裝舊版 Uber

  1. 打開 MuffinStore（或 DowngradeApp）
  2. 搜尋「Uber」
  3. 選擇支援 iOS 15 的舊版本
  4. 登入 Apple ID 驗證後下載安裝

5 鎖定 Uber 版本

  1. 打開 TrollFools
  2. 找到 Uber → 長按 → 點「鎖定版本」
  3. 回到 TrollStore →「重建圖標緩存」
  4. 前往「設定 → App Store」→ 關閉「App 更新」

🌐 備用方案：Uber 網頁版

如果舊版 Uber App 無法正常叫車，可直接用網頁版：

打開 Uber 網頁版 m.uber.com · 不需安裝 App 打開 Uber Eats 網頁版 ubereats.com · 瀏覽器直接點餐

⚠️ 重要注意事項

  1. 絕對不要升級 iOS，否則 TrollStore 將永遠無法使用
  2. 所有下載連結均來自 GitHub 官方，安全可靠
  3. 舊版 Uber 可能因伺服器限制無法叫車，請先測試
  4. 建議安裝後立即用 TrollFools 鎖定版本
  5. 如果 TrollStore 圖標消失，打開「提示 (Tips)」App 即可恢復
